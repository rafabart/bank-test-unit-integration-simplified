Projeto demo da incubadora. CRUD de Conta e Cliente.

Usando:

* JDK 11
* Springboot 2.2.1
* IntelliJ
* Rest
* H2
* Lombok
* Devtools
* Gradle

Para testes unitários e de integração foram usados:

* JBacon
* H2
* Junit4
* Junit5
* Rest-assured

Banco de Dados em memória, não necessita configuração.

URL de acesso a api:

```http://localhost:8080/```
